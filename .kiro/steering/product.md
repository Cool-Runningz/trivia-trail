# Product Overview

This is a Laravel React Starter Kit - a full-stack web application that provides a complete authentication system with modern UI components.

## Key Features

- User authentication (login, registration, password reset)
- Two-factor authentication with QR codes and recovery codes
- Email verification
- User profile management
- Password management
- Dark/light theme support
- Responsive design with mobile navigation

## Target Use Case

This starter kit is designed for developers who want to quickly bootstrap a Laravel application with React frontend, complete with authentication flows and modern UI patterns. It serves as a foundation for building SaaS applications, dashboards, or any web application requiring user management.

## Authentication Flow

The application uses Laravel Fortify for backend authentication and Inertia.js to bridge Laravel with React, providing a seamless SPA experience while maintaining server-side routing and validation.
